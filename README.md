# 📄 Arabic RTL & Docx Automation Engine

A professional Python-based web application designed to automate Microsoft Word document formatting, with specialized support for **Arabic (RTL)** scripts and pattern-based styling.

https://arabic-rtl-docx-automator.onrender.com/

---

## 🚀 Purpose
This project solves the common challenge of handling bidirectional text and inconsistent manual formatting in professional reports. It provides a seamless, web-based interface for instant document standardization.

## ✨ Key Features
- **RTL Support:** Custom **OOXML** injection to handle Arabic characters (\u0600-\u06FF) and force correct alignment.
- **Automated Styling:** Uses **Regex** to identify and apply Heading 1 and Heading 2 styles automatically.
- **Standardized Output:** Enforces **Arial 11pt** and **Justified** alignment for all body text.
- **Web Interface:** Built with **Streamlit**, allowing users to upload and process any .docx file without installing Python locally.

## 🛠️ Technical Stack
- **Language:** Python 3.13
- **Framework:** Streamlit
- **Core Library:** `python-docx` (with low-level XML manipulation)
- **Deployment:** Integrated for **Render** or Streamlit Cloud.
